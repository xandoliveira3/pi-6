# CounterApp - Aplicação React Native

Uma aplicação simples de contador feita com React Native e Expo.

## Funcionalidades

- Botão **+** para incrementar o número
- Botão **-** para diminuir o número
- Display do contador centralizado

## Como rodar

### Pré-requisitos

- Node.js instalado
- Expo CLI (`npm install -g expo-cli`)

### Passos

1. Instale as dependências:
```bash
npm install
```

2. Inicie o projeto:
```bash
npm start
```

3. Escaneie o QR code com:
   - **Expo Go** (Android/iOS)
   - Ou pressione `a` para Android emulator, `i` para iOS simulator
